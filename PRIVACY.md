# Cortex Privacy Policy

Cortex collects no personal data and sends nothing to external servers.

## What stays on your device

- AI provider usage statistics
- Conversation history cache (cleared on extension uninstall)
- Memory facts and project profiles
- API keys (stored in local browser storage only)

## What we never do

- Send your conversations to any server
- Track your browsing activity
- Share data with third parties
- Store anything outside your browser

## API key usage

If you configure an API key (Claude, OpenAI, or Gemini), it is used only to
call that provider's API for local memory extraction. The key is stored in
`chrome.storage.local` and is never transmitted to Cortex or any other service.
All processing happens locally in your browser.

## Data retention

All data is stored in your browser's local storage and IndexedDB. Uninstalling
the extension removes all stored data.
